Adds a CoffeeScript brush to SyntaxHighlighter Evolved.

== Description ==

This plugin easily adds a [CoffeeScript brush by Rod Vagg](http://rod.vagg.org/2011/02/coffeescript-brush-for-syntaxhighlighter/) to [SyntaxHighlighter Evolved](http://wordpress.org/extend/plugins/syntaxhighlighter/).

To use the brush, you can use `[coffee][/coffee]` or `[coffeescript][/coffeescript]`, just as with the other brushes.

== Installation ==

To install this plugin, place it in `/wp-content/plugins/` or isntall it through the search. Then activate it.
